hdtv.hsluv

This module is generated by transpiling Haxe into Python and cleaning the resulting code by hand, e.g. removing unused Haxe classes. To try it yourself, clone https://github.com/hsluv/hsluv and run:

haxe -cp haxe/src hsluv.Hsluv -python hsluv.py
