GetTH2(path)

Load a 2D histogram (``matrix’’) from a ROOT file. This can be either a ``true’’ TH2 histogram or a THnSparse of dimension 2.

Note: Unlike RootGet(), this function does not support shell-style pattern expansion, as this make it too easy to undeliberately load too many histograms. As they use a lot of memory, this would likely lead to a crash of the program.
