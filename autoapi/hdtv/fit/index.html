class hdtv.fit.Fit(fitter, color=None, cal=None)

Bases: hdtv.drawable.Drawable

Fit object

This Fit object is the graphical representation of a fit in HDTV. It contains the marker lists and the functions. The actual interface to the C++ fitting routines is the class Fitter.

All internal values (fit parameters, fit region, peak list) are in uncalibrated units.
